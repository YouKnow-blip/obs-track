# Релей OBS Remote Control

Это маленький сервер-ретранслятор. Он работает в облаке (Railway) и
связывает **сервер стримера** и **клиент оператора** по «коду комнаты».
Благодаря этому **не нужно пробрасывать порты на роутере и не нужен VPN**.

## Как это работает

```
Сервер стримера  ───►  /комната/host    ┐
                                       ├─►  Релей (Railway)
Клиент оператора ──►  /комната/client  ┘
```

- Обе стороны сами подключаются к релею (исходящие соединения), поэтому NAT/роутер не мешают.
- Релей **не знает токена** и просто пересылает сообщения. Проверка токена — на сервере стримера.
- Комната — любая строка (например, `my-stream-7421`). На сервере и клиенте она должна совпадать.

## Быстрый старт (Railway)

См. подробную инструкцию в `../docs/railway-relay.md`.

Коротко:
1. Зарегистрируйтесь на https://railway.app
2. Создайте новый проект и загрузите туда содержимое папки `relay/`.
3. Railway сам установит `websockets` и запустит `python relay.py`.
4. В разделе **Settings → Networking** нажмите **Generate Domain**.
5. Скопируйте выданный домен и используйте его как `wss://...up.railway.app` в настройках.

## Локальный запуск (для проверки)

```bash
cd relay
pip install -r requirements.txt
PORT=8080 python relay.py
```
Адрес для локальной проверки: `ws://localhost:8080`.
